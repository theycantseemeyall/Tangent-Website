## About
Tangent is a web-based study dashboard designed to help students
track their progress across multiple subjects and topics.

It includes:
- File explorer style ui
- Pomodoro Timer
- To-Do list
- Grade Tracker
- Calendar
- Smart Revision
- Live Clock

## Built With

- HTML
- CSS
- JavaScript
